hi vk,
i assume u too like me are a student and you need to trim a command line input for some project (like a shell) ''. Here is a very simple algo that will help u make a trim() function of ur own, incase u are not able to then the code is given here with. Make use of inbuilt functions like isspace() declaired in <ctype.h>. To begin with, since you are changing the whole string and wont be using the original string again, its wise to use pointers and get rid of the return type. Else you can copy the string to an array and do the needful.

ALGORITHM (pseudo code)
string trim (string S)
// input : a string S with any no of spaces or tabs etc
// output: the same string S with all the spaces removed

initialize a pointer P to the begining of string S
while P has not reached end of string
while *P is a space tab etc
move the next (P+1 to '\0') elements form P+1 to P
increment P
retrun S // not needed i have put cause it meets ur questions req

 char * trim (char *str)   // char * return type not needed

    char *p=str,*q;

          {
            q=p;

                   q++;
               }

       }

but there seems to be an obvious problem with it, its not readable and untill u have specific delimiters to observe (like >, < ,| etc ), this version of the function serves no purpose.

inorder to replace any no of spaces or tabs with a single space here is the code. all we have done is use a falg to tell if we have already replaced an entry with a single space or not!

Code:

#include<ctype.h>

char *trim (char *str)
{
  char *p=str,*q;
  int flag=0; // indicates no substitution has been done yet.
  if (*p=' ') flag =1; //dont spare 1st group of spaces (common bug)
  while(*p != '\0')
   {
     if (isspace(*p) && ! flag)
         {
            if (*p != ' ') // if *p is not a space but a \t, \n etc
               *p=' ';     // replace it with a space
            p++;
            flag = 1;
         }
      while ( isspace(*p) && flag )// shifting part
         {
           q=p ;
           while (*q != '\0')
              {
                 *q = *(p+1);
                 q++;
              }
         }
       flag=0;
       p++;
   }
  return str;
} 



sample OUTPUT:
input : \t this \t \t site \thas good \t questions \t\t !
output :this site has good questions !

Ahh, now this looks like a more readable string!

[color=darkblue]/* I assume that trim is always passed a non-NULL argument */

char *trim(char *s)
{
    char *p, *ps;
   
    for (ps = p = s; *s != '\0'; s++)
        if (!isspace(*s))
            *p++ = *s;

    *p = '\0';
    return ps;
}[/color]
