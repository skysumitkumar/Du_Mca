Question 0 : 
	In inheritance as we have studied it, private members of 
        the base class:

	a) Aren't inherited
	b) Are inherited but there is no way to access them in the
	   derived class

	   functions of the base class which are either public or

Question 5 : 
        How can we initialize private base class variables from a derived 
	  class constructor?

        a) We can assign values to the variables directly in the
	  derived class constructor
        b) We can call the base class constructor inside the {  }  
	  of the derived class constructor
        c) We can call the base class constructor after the function
	  header of the derived class constructor, with a : in-between 
	  the two

	  variables from a derived class constructor

   d (we will award credit for C, but note that we can not initialize private data in the dervied
