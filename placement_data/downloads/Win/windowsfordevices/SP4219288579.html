Smart Device Development -- A Complete Solution

Microsoft provides a complete smart device development solution consisting of two key components: Microsoft .NET Compact Framework, and Microsoft Visual Studio .NET

The .NET Compact Framework provides a set of runtime services and class libraries on smart devices enabling you to build rich applications quickly that are also robust and reliable. Visual Studio .NET complements this with a set of design-time tools to target the .NET Compact Framework, bringing a familiar RAD experience to mobile devices.


Smart Device Development Complete Solution


What is the Microsoft .NET Compact Framework?

The Microsoft .NET Compact Framework is the smart device development platform for Microsoft .NET, bringing the world of managed code and XML Web services to devices. The .NET Compact Framework greatly simplifies the process of creating and deploying applications to mobile devices while also allowing you to take full advantage of the capabilities of the device. The .NET Compact Framework is a compatible rich subset of the .NET Framework, thus providing the same benefits. But, it is designed specifically for resourceconstrained devices, such as PDAs and smart mobile phones.

Why Visual Studio .NET?

Visual Studio .NET provides first-class support for smart device development. You now have a single tool for device, desktop, and server development. This single tool simplifies development for those working on projects that span devices and desktops or servers. It also greatly reduces the learning curve for developers who have used Visual Studio .NET and are now moving into smart device development.
