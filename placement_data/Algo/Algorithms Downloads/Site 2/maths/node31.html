Insertion Sort

Once more we want to sort a list (x1,..., xn) into ascending order. Let me state the full algorithm directly

algorithm insertionsort
begin
  for i= n-1 down to 1 begin
    temp = xi
    j = i+1
    while(j <= n and xj < temp) begin
       xj - 1 = xj
      j=j+1
    end
    xj - 1 = temp
  end
end
