Question 3 (sketch)

This is a graph problem. The ADT to chose is DIGRAPH. Cars are nodes and the fact that car A has the same color as car B is represented by two directed edges from A to B and from B to A.

To find the number of colours, go through the nodes and if the node has not been visited, add 1 to the number of colours and call depth first search from this node. Complexity is O(n+m) where n is the number of nodes and m is the number of edges.
