An Implementation of the Euclidean Algorithm

An alternative way to see how this, and subsequent algorithms behave is to try out an implementation written to show the steps involved. By trying it with different inputs, you can get an idea of efficiency, and also check your understanding of what happens with unusual input. Here is an opportunity to try out the the Euclidean algorithm. This implementation is in Java, but the code is almost identical to the above. Both u and v can be changed; just edit the box, and press return within it. The new values shown will then be used in the calculation the next time the Run button is pressed.
