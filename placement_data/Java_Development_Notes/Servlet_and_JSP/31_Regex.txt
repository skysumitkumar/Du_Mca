[abc] it contain either a,b or c
[^abc] it contain character except a,b or c
[a-z] it start from small letter
[A-Z] it start from Capital letter
[a-zA-Z] it start from either a to z or A to Z
[0-9] it start from o to 9
[   ]? occurs 0 or 1 times
[   ]+ occurs 1 or more times
[   ]* occurs 0 or more times
[   ]{n} occurs n times
[   ]{n,} occurs n or more times
[   ]{y,z} occurs atleast y times but less than z times
\d	[0-9]
\D	[^0-9]
\w	[a-zA-Z_0-9]
\W	[^a-zA-Z_0-9]
