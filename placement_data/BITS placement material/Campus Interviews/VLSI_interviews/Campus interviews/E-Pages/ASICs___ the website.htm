Introduction to ASICs... the website

ASICs... the website is a project in computer-assisted education for integrated-circuit design.

  • ASICs... the website grew from a book: Application-Specific Integrated Circuits or ASICs... the book.
  • ASICs... the website now spans several thousand web pages and occupies several gigabytes of disk space.
  • ASICs... the websiteis large enough that you will probably need a copy of ASICs... the book in order to find your way around.
  • ASICs...the websitehas a plain look for two reasons: It is faster for students to access by modem from home and it is easy to project these pages for use in class by increasing the default font type and size.
  • ASICs...the websiteis a research project not a commercial product and some pages use advanced features that not all browsers support.

ASICs... the code

The links below are to files that I washed: that is I removed all character formatting (bold and so on), and replaced all nonbreaking spaces (&nbsp;) and tabs (displayed as triangular, delta signs in BBEdit for example). I replaced a nonbreaking space (&nbsp;) with a single space and replaced a tab with multiple spaces (either two or three depending on which looked best). I minimized additional editing in an attempt to keep the code correct (in a very few cases I deleted extra spaces, introduced by the absolute nature of FrameMaker tabs, that caused the lines to be excessively long). Adobe PageMill, Adobe FrameMaker, and Internet browsers from both Netscape and Microsoft handle tabulation marks (tabs) and tab spacing in different ways. This means the code in the HTML files below (which do not contain tabs) may not always align in your browser and may not match exactly the appearance in ASICs... the book. You should be wary of characters in any file copied from HTML that look like spaces (a nonbreaking space, or fixed-width space, for example) but that some software (VeriWell is an example) will not accept as a legal space character. When in doubt you should paste the code into an application that will display the difference between regular and special space characters (Microsoft Word has this feature).

Errors in ASICs... the book

We (the readers and I) have found the following errors (what a publisher calls "an error of fact"). Interestingly, many of the typography errors (a comma being in Courier font instead of Times font, for example) have surfaced as I convert the text to HTML. I am making a list as I collect these. I welcome your comments. Please email me if you find an error.

 Page  Error of fact and/or correction
 40  A p-channel transistor requires a logic '0' (again from now on, I'll just say a '0') on the gate to make the switch conducting (to turn the transistor on). The p-channel transistor has a buble on its gate to remind us that the gate has to be a '1' to turn the transistor off.
   Readers have found about six more errors of fact, when I get a chance I will include those here and credit the finders.

You can tell the version (printing) of a book by looking inside the front cover. You will usually see a list of digits (1 through 10). If digits 1 and 2 are missing you have the third edition, and so on. Some publishers hide this information more than others. Addison-Wesley also prints the edition infomation explicitly on the inside cover page.

Publishers tend to print small runs of books often these days instead of one big initial run as they used to. This has to do with changes on the rules on write-offs from inventory by the IRS. As a very rough guide, for most engineering books, the publisher will print about 1000&SHY;2000 copies per printing. This gies you a very rough idea of how many copies a book sells.
