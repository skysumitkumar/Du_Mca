Describes the Office 2003 Primary Interop Assemblies (PIAs) redistributable that contains the Primary Interop Assemblies for Office 2003 products.
•Description of the Security Update for Office 2003: WordPerfect 5.x Converter: September 14, 2004
Describes a vulnerability that may allow arbitrary code to run when the system uses the converter to open a maliciously crafted document. The update resolves this vulnerability so that the files that the converter opens are handled. appropriately.
•Description of the Update for Office 2003: February 8, 2005
Describes update that increases the reliability of Smart Tags by additionally restricting the way that Web sites are associated with the Smart Tag actions.
•"There are no Office Assistant character files present on the system" error message when you start an Office 2003 program

Describes a problem where the Office Assistant starts in Office 2003 and the animations for the assistant are turned on after you click to clear the "Menu and window animation" check box in Remote Desktop Connection in Windows Server 2003.
•Generic icons appear in the Web Collections folder of Clip Organizer in an Office 2003 program
Describes a problem where you may see generic icons instead of thumbnails in the Web Collections folder in Clip Organizer in an Office 2003 program.
•The Office 2003 spelling checker incorrectly identifies the word "Iqaluit" as misspelled
Describes a problem that occurs if you type Iqaluit. A hotfix is available to resolve this problem.
•A TIFF image is not scaled to print on the whole page when you use the "Fit to page" print option in the Microsoft Office Document Imaging program in Office 2003
Describes a problem where a TIFF image is not printed to the size of the page when you use the "Fit to page" print option in the Microsoft Office Document Imaging program in Office 2003.
•You receive an "Another version of this product is installed" error message when you try to help protect or to open an IRM document in an Office 2003 program
Describes a problem where you receive a "Another version of this product is installed" error message when you try to open an IRM protected document or you try to protect a document in an Office 2003 program.
