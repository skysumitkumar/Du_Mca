Troubleshooting - Microsoft Exchange Server 2003

•You receive an "A connection could not be made to the remote directory service...ID no: c1030b11" error message when you try to install a Site Replication Service in Exchange Server 2003 or in Exchange 2000 Server
Describes an issue that occurs if you specify an incorrect Exchange service account password when you install the Site Replication Service.
•The Internet Address template does not appear correctly after you upgrade to Exchange Server 2003
Describes a problem where the Traditional Chinese Internet Address template is not updated correctly when you upgrade to Exchange Server 2003 from Exchange 2000 Server.
•Outlook Web Access clients experience problems in a mixed-mode Exchange environment when compression is enabled

•Response text does not appear in Outlook or in Outlook Web Access when you accept or decline a meeting request by using CDO with Exchange Server 2003
Explains that when you use Collaboration Data Objects to accept or to decline a meeting request in an Exchange Server 2003 environment, the response text does not appear when the meeting response is viewed in Outlook or in OWA. Includes a workaround.
•Outlook Mobile Access users cannot access their mailboxes on a heavily-loaded Exchange Server 2003 computer
Describes a problem that may occur if Exchange Server 2003 is running on a heavily loaded Windows 2000-based computer. Contains a resolution that requires a hotfix. Contains a workaround.
•The Exchange Server 2003 Recipient Update Service modifies an imported user's proxy address when you configure an interorganizational connection agreement
Describes an issue where the proxy address of users from the source organization is modified by the Recipient Update Service.
•An SMTP connector does not pick up outgoing security settings after you configure it in Exchange Server 2003

Describes a problem that occurs after the primary SMTP address is changed in the Exchange default recipient policy.
•"ID no: c0070424" error message when you try to create the Exchange System Attendant resource on an Exchange Server 2003 cluster node
Explains a behavior where you cannot create the Exchange System Attendant resource on a cluster node if an Exchange resource type is missing from the "Resource type" list. You must reinstall Exchange Server 2003 to resolve the behavior.
