SUMMARY

Microsoft has released a service pack for Microsoft Office Outlook 2003 with Business Contact Manager. The following "Article contents" section describes information that this article contains about the service pack.
Back to the top

INTRODUCTION

Microsoft Office Outlook 2003 with Business Contact Manager Service Pack 1 (SP1) provides the latest updates to Outlook 2003 with Business Contact Manager. Outlook 2003 with Business Contact Manager SP1 contains significant usability enhancements and stability improvements. Some of the fixes that are included with Business Contact Manager SP1 have been previously released as separate updates. This service pack combines the previously released updates into one update.

Service pack details

List of issues that are fixed in the service pack

Outlook 2003 with Business Contact Manager SP1 fixes the issues that are described in the following hotfix packages:
831747 Overview of the Business Contact Manager for Outlook 2003 update: Windows Small Business Server 2003
842496 Description of the Business Contact Manager for Outlook 2003 Security Update: June 8, 2004

Outlook 2003 with Business Contact Manager SP1 fixes the following issues that were not previously documented in the Microsoft Knowledge Base.

You are not warned when you open an e-mail profile that contains Exchange as a service


When you open an e-mail profile that contains a stand-alone Exchange service in Outlook 2003 with Business Contact Manager, you do not receive a warning message that indicates that Business Contact Manager is not compatible with profiles that contain an Exchange service.

This update to Outlook 2003 with Business Contact Manager adds the following warning. Therefore, if you open an e-mail profile that contains an Exchange service, you receive the following warning message:
Business Contact Manager will not work with this Outlook profile. The profile uses a version of Microsoft Exchange server that Business Contact Manager doesn’t support.

For information about supported versions of Exchange server, see Use Business Contact Manager with Exchange server at http://office.microsoft.com.


You receive a "Microsoft Office Outlook has encountered a problem and needs to close" error message when you work in Microsoft Office Outlook 2003


When you work in Outlook 2003, you may receive an error message that is similar to the following:
Microsoft Office Outlook has encountered a problem and needs to close. We are sorry for the inconvenience.
Please tell Microsoft about this problem.
We have created an error report that you can send to help us improve Office Outlook. We will treat this report as confidential and anonymous.
To see what data this error report contains, click here.
