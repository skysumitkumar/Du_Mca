Question:
Make an equilateral diamond, then inscribe a square inside the center of the diamond so that each corner of the square touches the midpoint of each segment of the diamond. Now draw two line segments so that each segment connects opposite corners of the diamond. Draw two line segments so that each segment connects opposite corners of the squares, How many triangles are in this figure?

Answer:
44 triangles

Using 1 "piece" - 16
Using 2 "pieces" - 16
Using 4 "pieces" - 8
Using 8 "pieces" - 4
