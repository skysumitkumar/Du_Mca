3D Cubes

index.htmlcubes.html

Try to fit the two cube groups (red and magenta) into each other to form a single 3x3 block of cubes. That is, try to move the pieces into position to form a single slab!
  • 1: Select Cube Group 1 (The red cubes)
  • 2: Select Cube Group 2 (The magenta cubes)
  • a: To select ALL Cube Groups (red and magenta cubes)
  • Arrow Keys: to move (translate) cubes
  • PageUp/PageDown: to rotate cubes around selected axis
  • x,y,z: to set the rotation axis to x,y,z respectively
  • s/spacebar: change speed to fast/slow
  • Mouse Click: to set origin of axes (rotation point)

  • Note: This game does not "end", you keep playing until you're satisfied, and then try it all over again by clicking on "New"! Good Luck!
