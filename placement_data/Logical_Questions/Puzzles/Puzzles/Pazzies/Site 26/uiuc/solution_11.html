Question:
How can you test a number for divisibility by 11

Answer:
Thanks to Dan Higgins for his explanation!

  1. Add the digits in odd number columns (ones, hundreds, ten-thousands, etc.)
  2. Add the digits in even number columns (tens, thousands, hundred-thousands, etc.)
  3. If the difference is divisible by 11, the number is divisible by 11.

Example:
Number: 22846193705192
1. 2+1+0+3+1+4+2=13
2. 9+5+7+9+6+8+2=46
3. 46-13=33. Divisible by 11 (22846193705192 / 11 = 2076926700472).
