Solution to puzzle 61: Two cubes?

If n + 3 and n2 + 3 are both perfect cubes then their product should also be a perfect cube.

But (n + 3)(n2 + 3) = n3 + 3n2 + 3n + 9 = (n + 1)3 + 8.

The only perfect cubes that differ by 8 are (-8, 0) and (0, 8).
However, (n + 1)3 + 8 = 0 yields n = -3, and (n + 1)3 + 8 = 8 yields n = -1.
For neither of these solutions are both n + 3 and n2 + 3 perfect cubes.

Therefore, if n is an integer, n + 3 and n2 + 3 cannot both be perfect cubes.

One cube?

For what integers n is 18(n2 + 3) a perfect cube?
