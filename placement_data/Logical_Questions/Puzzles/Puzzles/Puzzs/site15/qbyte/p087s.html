Solution to puzzle 87: 2004

Evaluate 22004 (modulo 2004).

Note that the prime factorization of 2004 is 22 × 3 × 167.
Now consider 212×166 21992 (mod 2004).

By Fermat's Little Theorem, 2166 1 (mod 167).
Hence 21992 = (2166)12 1 (mod 167).

Further, 21992 = (22)996 1996 1 (mod 3).

By the Chinese Remainder Theorem, since 3 and 167 are coprime, there is one solution (modulo 3 × 167) for each pair of solutions, modulo 3 and 167.  Hence, in this case, there is one solution, modulo 501.

We have x 1 (mod 167) and x 1 (mod 3).
From the first congruence, x = 1 + 167t, for some integer t.
Then x = 1 + 167t 1 (mod 3).
Hence 167t 2t 0 (mod 3), and so t 0 (mod 3).
Setting t = 3s, we obtain x = 1 + 501s, for some integer s.
That is, x 1 (mod 501.)

Similarly, since 21992 = (22)996 0 (mod 4), we have x = 1 + 501s 0 (mod 4), from which s 3 (mod 4).
Hence x = 1 + 501(3 + 4r) = 1504 + 2004r, for some integer r.
That is, 21992 1504 (mod 2004).

Therefore, working modulo 2004,  22004  = 21992 × 212
  1504 × 212
  = (1504 × 22) × 210
  4 × 1024
  88.
