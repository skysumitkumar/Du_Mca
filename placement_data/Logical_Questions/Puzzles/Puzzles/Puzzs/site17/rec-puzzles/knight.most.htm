Solution to the /competition/games/chess/knight.most problem

n^2/2 for n even >= 4.

Divide the board in parts of 2x4 squares. The squares within each part are paired as follows: 

-----
|A|B|
-----
|C|D|
-----
|B|A|
-----
|D|C|
-----
Clearly, not more than one square per pair can be occupied by a knight.
