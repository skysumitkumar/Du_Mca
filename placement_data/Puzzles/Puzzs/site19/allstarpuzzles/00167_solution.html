Word Square 6
DUTY calls.
May 30, 2005
From the introduction, each row and each column of Word Square 6 contains the characters DUTY and a blank cell in some order, so that no character or space can be repeated in a row or a column. By clue 1, U is the 3rd letter across row 1, so U is in column 3 or column 4. T is the first entry down column 3 (clue 5), so U can't be in row 1 of column 3 and is in row 1 of column 4. Neither Y (8) nor T (10) is 1st down column 2. If U were 1st down column 2, there would be a space in row 1 and the U would be in row 2--a conflict with clue 11. So D is the 1st letter down column 2. If the space in column 2 were in row 1, D would then be in column 2, row 2--the 2nd letter across the row, no (7). D is in row 1 of column 2. If in clue 5 the T in column 3 were in row 1, by clue 1, the space in row 1 would be in column 1 and the Y in row 1 in column 5. By clue 9, the U in column 1 would be in row 3. By clue 6, the U is the 3rd letter down column 5; the U would be in row 4, with the D in column 5 in row 5 (3). If the T in column 5 were in row 3 with the space in row 2, by clue 7, the D in row 2 would be in column 3; the U in row 2 would have to be in column 2, the 2nd letter down that column--a conflict with clue 8. If the T in column 5 were in row 2 with the space in row 3, by clue 8, the U in column 2 couldn't be 2nd and would be in row 5, with the U in column 3 in row 2. The D in row 2 would be in column 4, and the D in row 3 would be in column 3. However, this would make the column 3 Y the 4th letter down the column--no (2). So, the T in clue 5 must be in row 2, column 3, with a space in row 1 of column 3. By clue 7, D in row 2 must be in column 4. By clue 9, U is 2nd down column 1. If U were in row 3 with a space in row 2, Y in column 2 would be in row 2 (8) and U in row 2 in column 5--impossible given clue 6. So, the U in clue 9 is in row 2 of column 1. Since D is the 3rd letter in row 2 (7), the space in row 2 must be in column 2. Then Y in row 2 is in column 5, putting T in row 1 in column 5 and Y in row 1 in column 1. By clue 8, Y in column 2 is in row 3. Since Y isn't the 4th letter down column 3 (2), the row 5 Y is in column 4; the Y in column 3 is in row 4. D is the 4th letter down column 5 (3). If the D in clue 3 were in column 5, row 4, with a space in column 5, row 5, U would be in column 5, row 3. Then U would be in in column 3, row 5 and a D in column 3 row 3. U would be in column 2, row 4 and T in column 2, row 5. D would be in column 1, row 5, next to the T, contradicting clue 4. So, the D in column 5 is in row 5. In column 3 D is in row 3 and U in row 5. Then there is a T in column 2, row 5, with the U in row 4. Row 5's space is in column 1. D in column 1 is in row 4 with T in row 3. T in column 4 is in row 4 and space in row 3. In column 5, U is in row 3 and space in row 4. In sum, Word Square 6 is as follows:
