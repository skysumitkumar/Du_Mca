7x7 Logi-Number Puzzles

Logi-Number Boxes are a cross between logic problems and mathematical puzzles. In each, you must determine which values each of the variables is equal to, using the rules of the game and the given clues.

The rules are that all the variables equal integer values between one and the number of variables in the puzzle, inclusive, and that none of the variables are equal to each other. For example, in a Logi-Number Box with six variables, each variable equals one of the whole numbers between one and six. Since a variable may not be equal to any other variable, all the values between one and six are used.

You may find it convenient to solve the puzzle with the aid of the grids provided. Put an 'X' in a box when you know a statement to be false (for example, if you know that A is not equal to 3, put an 'X' in box A3) and put a '.' or check mark to note truthful statements.

The symbols used on this page have the usual mathematical meaning, for the most part. Note, however, that the '!=' operator means "does not equal to" in accordance with many programming languages. (There are no factorial operations in these puzzles, which the '!' symbol is normally used to denote.)

There is only one unique answer per puzzle. You may wish to print this page out and work on paper.

If you have trouble solving a puzzle, the corresponding Hint link will help you out.
