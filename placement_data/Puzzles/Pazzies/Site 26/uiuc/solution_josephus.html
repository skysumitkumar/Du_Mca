Solution

Question:
Consider n people (1...n) placed in a circle. Starting from person 1, you have to remove every second person. Thus, first you would remove person 2, then 4, and so on. The question is, given n, how do you determine who will be the last person left?

Answer:
Thanks to John Tripp for the explanation.

Follow these steps:

  1. Represent n as a binary number,
  2. Eliminate leftmost digit (essentially subtracting highest power of 2 in n)
  3. Multiply remainder by 2
  4. Add 1. This gives you the number of the last person left.

Examples.

n=3:

  1. 11
  2. 01
  3. 10
  4. 11
Answer = 3

n=14:

  1. 1110
  2. 0110
  3. 1100
  4. 1101
Answer = 13

n=128:

  1. 10000000
  2. 00000000
  3. 00000000
  4. 00000001
Answer = 1
