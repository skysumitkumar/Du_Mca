SUMMARY

Microsoft has released a service pack for Microsoft Office 2003 that contains fixes for Microsoft Office Outlook 2003. This article describes the following items about the service pack:

INTRODUCTION

Microsoft Office 2003 Service Pack 1 (SP1) provides the latest updates to Microsoft Office 2003. Office 2003 SP1 contains security enhancements, and stability and performance improvements. Office 2003 SP1 also includes many performance and feature enhancements to Microsoft Office Outlook 2003.

For additional information about Office 2003 Service Pack 1, click the following article number to view the article in the Microsoft Knowledge Base:

List of issues that this service pack fixes

Office 2003 SP1 fixes the issues that are described in the following hotfix packages:

830003 Description of the Outlook 2003 hotfix package: December 17, 2003
833984 Description of Outlook 2003 hotfix package: January 7, 2004
833986 Description of the Outlook 2003 hotfix package: January 17, 2004
833856 Description of the Outlook 2003 hotfix package: January 20, 2004
834714 Description of the German version of Outlook 2003 hotfix package: January 29, 2004
835210 Description of the Outlook 2003 hotfix package: March 2, 2004
837019 Description of the Outlook 2003 hotfix package: March 17, 2004
838020 Description of the Outlook 2003 hotfix package: April 7, 2004
834716 Description of the Outlook 2003 hotfix package: May 20, 2004


Office 2003 SP1 fixes the following issues that were not previously documented in the Microsoft Knowledge Base.

The envelope part of an e-mail message is missing when you send multiple e-mail messages from an Office 2003 program


In an Office 2003 program, if you try to create multiple e-mail messages, the e-mail header (named the envelope) the contains an e-mail toolbar may be removed from the e-mail messages.


The Public folder home page does not appear when you work off-line


The Public folders home page may not appear as expected in Outlook 2003 when you work off-line even if the following conditions are true:
•The public folder has been added to your Favorites list.
•Both the home page and the public folder have been selected for synchronization.
In some cases, you may also receive the following message:
Unable to display folder.




The Business 2 field that is in a contact that is located in the GAL is not copied to a personal address book (PAB)


When you try to copy a contact that is located in the Global Address Book to a PAB, the Business 2 field that is in the contact is not copied.


You cannot find the German Eszett character (ß) by using the Find feature


When you try to find a word that contains the German Eszett character (ß) by using the Find feature in Outlook 2003, the character is replaced with "ss" characters. Additionally, when you try to find a word that contains "ss" characters, the "ss" characters are replaced with the Eszett character. This issue occurs because both the Eszett character and the "ss" characters are treated as the same characters in the Look for list.

For example, when you try to find the word Keßler, you find the word Kessler instead.


The GoToDate method with Late Binding causes Outlook to quit unexpectedly


When you use the GoToDate method with Late Binding in a Microsoft Visual Basic for Applications (VBA) macro, Outlook 2003 may quit unexpectedly, and you receive the following error message:
Run-time error: Method 'GotoDate' of object 'View' failed.


Nothing indicates that an attachment is included in a printed HTML e-mail message


When you print an HTML e-mail message that has an attachment, there is no field or icon for attachments. This hotfix adds an Attachments box to the HTML e-mail message. This box displays the file name of the attachments.


Smart Tags do not function after you install Microsoft Windows XP Service Pack 2


After you install Windows XP Service Pack 2, Smart Tags in an Outlook 2003 e-mail message do not function.


A delegate receives the "Can't open the item" error message when the delegate opens a manager's calendar


When a delegate opens a manager's calendar, the delegate receives the following error message at the time that they open the calendar or 15 minutes after they open the calendar:
Can't open the item. Unable to open the free/busy information.


File name extensions that you cannot open in Windows XP are sent as attachments in Outlook 2003


After you install Windows XP Service Pack 2, some file name extensions that cannot be opened in Windows XP may be sent as an attachment in Outlook 2003.


You can unexpectedly create a new non-e-mail folder in the MSN Connector folder


After you set up an MSN Connector account in Outlook 2003, you can unexpectedly create a new non-e-mail folder in the MSN Connector folder.


The body of an e-mail message is blank when you send an updated meeting request


When you send an updated meeting request that contains text in the body of an e-mail message, the body of the received updated meeting request is blank. This issue can occur if the meeting request did not originally contain text in the body, and if it was originally created with Outlook Web Access.


Outlook 2003 quits unexpectedly when you start it on a computer that is running the Turkish version of Microsoft Windows


When you try to start Outlook 2003 on a computer that is running the Turkish version of Windows, Outlook 2003 may quit unexpectedly.


A full download of address book always occurs in Outlook 2003


When Outlook 2003 updates the Offline address book, a full download of the address book occurs.


English-language content appears when you click a link that is in a welcome message in a non-English version of Outlook 2003


When you click a hyperlink in the welcome message in a non-English version of Outlook 2003, English-language content appears.


Outlook 2003 rules are missing after you create a new rule in OWA


If you create a new Outlook 2003 rule in Outlook Web Access, many of your previous rules may be missing.


The body of an HTML item is missing when you insert it in an Outlook 2003 e-mail message


When you insert an HTML item in an Outlook 2003 e-mail message, only the header of the HTML item is inserted.


The Outlook Junk E-mail Filter may not correctly filter the e-mail message


If an e-mail message contains a HTML MIME part, the Outlook Junk E-mail Filter may not correctly filter the e-mail message.


Completed tasks also print when you print the Taskpad with the calendar


When you print the Taskpad with the calendar, the completed tasks print together with your active tasks.


Outlook 2003 times out when you download e-mail messages from multiple Post Office Protocol version 3 (POP3) e-mail accounts


When you have multiple POP3 e-mail accounts configured in your Outlook profile that exist on the same server, Outlook 2003 may time-out when you try to download e-mail messages.


Duplicate e-mail messages are downloaded from POP3 accounts that supports UIDL or TOP


If you download e-mail messages from a POP3 account that supports UIDL or TOP, some e-mail messages may not be deleted after downloading. Therefore, you may download some duplicate e-mail messages.


Microsoft Office Internet Free/Busy service is deactivated in Office 2003 SP1


The Office Internet Free/Busy service has been deactivated in Office 2003 Service Pack 1.


Increased polling of e-mail server after you install update to the Outlph.dll file


After you install an Outlook 2003 hotfix package that contains an updated Outlph.dll file, polling sent by Outlook 2003 may be increased. This causes a heavier than expected load on the e-mail server.


Deleted e-mail messages are not removed from a Pocket PC or Smartphone device


When a Pocket PC or Smartphone device is in the USB cradle, and the user synchronizes the device with Outlook 2003 by using ActiveSync, a deleted e-mail message is not removed from the device when Outlook 2003 is running in Cached Exchange Mode.


Outlook starts in Personal Information Management (PIM) mode after you cancel the Configuration Run Wizard


If you cancel the Configuration Run Wizard after you install Outlook 2003, Outlook 2003 will start in PIM mode. Therefore, you have no e-mail functionality.


E-mail that is from a sender that is on the same e-mail server may be classified as junk e-mail


When e-mail is delivered to you from a sender that is on the same e-mail server, the e-mail may be misclassified as junk e-mail. This issue may occur when you use the Safe Lists Only option on the Options tab of the Junk E-mail Options dialog box in Outlook 2003.


The Out of Office (OOF) rule does not fire in Outlook 2003 in Cached Exchange Mode



If you create a client-side OOF rule that creates a message, moves the e-mail message to a personal files (.pst) folder, or performs a custom action, the OOF rule does not fire if you run Outlook 2003 in Cached Exchange Mode.


E-mail messages that are based on an organizational form do not appear correctly in Outlook 2003 in Cached Exchange Mode


An e-mail message that was created from an organizational form may not appear as expected when Outlook 2003 is in Cached Exchange Mode. Additionally, your connection to the e-mail server is lost.


Cannot report separate foreground and background remote procedure calls (RPC) calls for Microsoft Exchange clients


When you monitor an e-mail client, both the foreground and the background RPC cannot be reported separately for Exchange clients.


Earlier revision of an organizational chart are sent as an e-mail attachment


When you create a new HTML e-mail message that contains an organizational chart and then save the e-mail message in the Drafts folder, a .GIF image of a previous version of the organizational chart is sent as an attachment with the sent e-mail message .


There is nothing to indicate that the Enable mail logging (troubleshooting) option is turned on in Outlook 2003


After you turn on the Enable mail logging (troubleshooting) option in Outlook 2003, there is nothing to indicatethat the option is turned on. If the Enable mail logging (troubleshooting) option is turned on, the performance of Outlook 2003 may decrease.


CPU usage is 100 percent when you download a full e-mail message on a slow connection


If you double-click an e-mail message header to download the full e-mail message, CPU usage may be 100 percent if you are using a slow connection.
