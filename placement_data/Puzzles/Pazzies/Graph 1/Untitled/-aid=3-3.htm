Troubleshooting - Microsoft Office Outlook 2003

•"Unable to Load Free/Busy Information" Error Message When You Try to Use Meeting Planner in Outlook 2003
When you try to use the meeting planner in Microsoft Outlook 2003 to view another person's Free/Busy information, you may receive an error message that is similar to the following: Unable to load Free/Busy information. This behavior may occur if...
•How to troubleshoot error messages that you receive when try to send and receive e-mail in Outlook and in Outlook Express
Lists various error messages that you may receive when you send or receive e-mail messages in Outlook and in Outlook Express. Discusses the possible causes and subsequent resolutions.
•You receive a "Could not find a contact with this e-mail address" error message when you use the "Look Up Contact" feature in Outlook 2003
Explains that you may receive a "Could not find a contact with this e-mail address" message when you use the "Look Up Contact" function in Outlook 2003. You can double-click the e-mail address to find the contact in the Global Address List.
•Names are not resolved as expected when the name cache is full in Outlook 2002 and Outlook 2003
Describes that Outlook takes a long time to display the drop-down list when you type a name from the cache. This problem occurs when the name cache is full. You can turn off the AutoComplete feature or remove some names to avoid the problem.
•You cannot connect to your MSN account in Outlook 2003
Explains that you receive an "(0x800CCC32) Verify your name and password" error message when you try to access MSN. This issue occurs when the privacy settings are set to High. Requires that you enable cookies from "hotmail.com" and "msn.com."
•You receive an error message when you try to open another user's folder even if you have owner permission in Outlook 2003
Explains that although you have the required permission to open another user's mailbox, you receive an "Unable to display the folder, cannot open the mailbox for this user" error message when you try to access the mailbox.
•You receive a "Microsoft Outlook could not sign or encrypt this message" error message when you send or read a signed e-mail message in Outlook 2003
Explains that you may not be able to send or receive a signed e-mail message or to encrypt or decrypt a message in Outlook 2003. Requires the system administrator to reissue your Key Management Services certificates.
•You receive a cannot reschedule error message when you try to change the time for a recurring meeting in Outlook 2003
Describes the "Cannot reschedule an occurrence of the recurring appointment <meeting_name> if it skips over a later occurrence of the same appointment" error message. This problem may be caused by your location relative to the International Date...
•You receive an "Netscape is unable to find the file or directory" error message when you try to add a SharePoint Team Services folder to Outlook 2003
Describes that an "Netscape is unable to find the file or directory" error message may occur when you add a SharePoint Team Services folder to Outlook 2003. Requires you to install the most recent version of Netscape to resolve the problem.
•You Receive an Error Message When You Use the Inbox Repair Tool to Scan a .pst File
When you use the Inbox Repair Tool (the Scanpst.exe file) to scan a personal folders (.pst) file for errors, you may receive the following error message, even if there are no errors in the file: Internal errors were found in this file. They must...
•You receive a "Can't open this item. Out of memory or system resources" error message when you try to open a note in Outlook 2002 or Outlook 2003
Explains that when an Outlook Note contains the maximum number of characters that Outlook allows, the note will not be opened, and you receive an error message.
•"The address list could not be displayed" error message when you use your Contacts to address a new message in Outlook
Explains that you may receive an error message when you try to address a new message from your Contacts in Outlook 2002 and in Outlook 2003. You must remove and reinstall the Outlook Address Book to resolve the issue.
•You receive a "The messaging interface has returned an unknown error " error message when you try to use Outlook to make a change to a recurring appointment in the Calendar of your Hotmail account or of your MSN e-mail account
Describes an issue where you receive an error message when you use Outlook to make a change to a recurring appointment in Hotmail or in MSN. The change does not appear when you view the calendar in your Web browser.
•You receive the "The operation failed. An object could not be found" error message when you click Send and Receive in Outlook 2003 and Outlook 2002
Describes that you may receive an error message when you click Send and Receive in Outlook 2003 and Outlook 2002. Requires you create a new profile to resolve the problem.
•You receive a synchronization cannot start error messages when you synchronize your Pocket PC with Outlook 2003
Discusses how you cannot synchronize your Pocket PC with Outlook 2003 when you are using ActiveSync 3.6 or earlier versions. To resolve this problem, you can update ActiveSync to 3.7 or later versions.
•You cannot use the /select switch to start Outlook 2003
Describes an error message that you may receive if you use the /select switch in command-line switches to start Outlook 2003.
•BUG: The Outlook shortcut on the Quick Launch bar is not updated when you upgrade from Outlook 2000 to Outlook 2003
Explains that if you click the shortcut for Outlook 2003 in the Quick Launch bar, you receive an "Outlook.exe file not found" error message. You can start Outlook by using the Start menu or from the shortcut on your desktop.
•Archived file data is lost if you use Windows Explorer to open an archive file from Outlook 2003
Describes a problem that occurs when you try to use Windows Explorer to open an archived Outlook 2003 .pst file. You may receive error messages and lose the archived data. Open the data file in Outlook to work around this issue.
•You receive an "Microsoft Outlook could not sign or encrypt this message because your certificate is not valid" error message when you send a signed e-mail message in Offline mode in Office 2003
Describes an error message that occurs when you try to reply to a signed e-mail message or when you try to send a new signed e-mail message in Offline mode. Requires you to download the full detail offline Address Book to work around this problem.
•You receive a "Form load failed" or "Path invalid" error message when you try to create a custom form and publish the form to a search folder in Outlook 2003
Explains that a "Form load failed" or "Path invalid" error message may appear when you create a custom form and publish the form to a search folder. You can publish your form to a standard folder to resolve the problem.
•Outlook 2003 does not recognize a comma as an e-mail address separator
Explains that Outlook 2003 does not accept a comma as an e-mail address separator. This behavior is by default. Provides steps that you can use to configure Outlook 2003 to recognize a comma as a valid separator in an e-mail address.
•You receive the "0x80004005 : The operation failed" error message when you receive new e-mail messages in Outlook
Describes an issue when you receive the "0x80004005 : The operation failed" error message when you receive new e-mail messages.
•You receive an error message when you exit or when you try to add WinFax PRO to Data File Management in Outlook 2003
Explains an error that occurs when you exit Outlook 2003 or when you send or receive messages. The error report indicates that the problem occurs in Msmapi32.dll. You may also receive an Outlook warning when you add WinFax PRO to Data File...
•Exchange Client Extensions events do not fire correctly in Outlook 2003
Describes a problem where an Outlook 2003 profile that is configured with IMAP may not fire Exchange Client Extensions events.
•You receive an error message when you use Outlook Express to access your Hotmail account
Explains that you may receive an "Unknown Error" error message or a "Hotmail no longer allows email access via Outlook Express for free Hotmail/MSN accounts" error message when you try to open your Hotmail account by using Outlook Express.
