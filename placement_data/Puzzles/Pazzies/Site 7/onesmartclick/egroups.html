Object Oriented Programming/Method/Patterns eGroups

Mostly 'PUBLIC' eGroups.

OOADandUML - Cool stuff (Newsletters) on Object Oriented Analysis and Design (OOAD), Hot tips on Unified Modeling Language (UML)

DpExplained - Design Patterns Explained is a discussion group moderated by Alan Shalloway, co-author of Design Patterns Explained A New Perspective on Object-Oriented Design. Its purpose is three-fold 1) provide a forum for developers using and learning design patterns, 2) provide assistance to readers of the Design Patterns Explained book, 3) be a part of Net Objectives Community of Practice.

OOTips - Object Orientation Tips is a free service devoted to object oriented technologies. Its purpose is to allow developers to keep up-to-date with object oriented technologies, without having to read through high volume, low signal-to-noise sources. I try to extract the most useful and interesting posts from other mailing lists, newsgroups and web forums, and post about one article every week or two. More info and past tips at http //ootips.org.

ITmethods - The ITmethods email discussion list has been established for IT professionals, developers, methodologists, and programmers to discuss topics related to IT methodology creation and utilization, CASE tools, object oriented programming, patterns, and software engineering. It is sponsored and moderated by the Methodology.org website.

LabToo - The list is used to share information about the state of the art of the object-oriented technologies, including subjects related with Mobile Agents, Data Bases, Human Computers Interaction design, Operating Systems, Compilers, Abstract Machines, etc. The list is also used to provide information about lectures, conferences, articles, call for papers and any other event related with the OO Technologies.

Argo-Dev - Technical discussion of development of ArgoUML The Open Source CASE Tool with Cognitive Support for Object-Oriented Development.

CincyObjectUsers - Cincinnati Object Oriented Technology Users Group focuses on OO software development languages, technologies, and processes. Topics include Agile Processes, Extreem Programming (XP) Practices, Java, Ruby, .Net, C#, XML, Object Databases, Distributed Object Architecture Development, Smalltalk, WebObjects, OO Design. Conference Reports from OOPSLA, Smalltalk Solutions, and XP Universe. Or make a suggestion!

OoDb - Discussion forum for object-oriented databases. Read the summary for a quick overview. A proposal for the S.O.D.A. API is available for download.

OoDEventHelixUpdate - EventHelix Update is a monthly newsletter about updates to EventHelix.com, a website dedicated to Realtime Software Development. We cover articles that will be of interest to software developers for Realtime Systems. The main areas of interest are o Realtime and Embedded System Development Basics o State Machine Design o Realtime and Embedded Design Patterns o Object Oriented Programming and C++ in Realtime and Embedded Software Development o Congestion Control o Hardware and Software Fault
