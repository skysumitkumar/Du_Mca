Antwoord op: Faites Vos Jeux

Noem de oorspronkelijke inleg b. De kans dat je in een ronde verliest is 18/37. De kans dat je n rondes achter elkaar verliest (en dan dus met verlies moet stoppen) is (18/37)n. De kans dat je een keer wint is dus 1-(18/37)n. Als je n rondes achter elkaar verliest, is je totale inzet (en verlies) (2n-1) × b. Als je een keer wint is je winst gelijk aan b. De verwachtingswaarde voor je winst is dus (18/37)n × ( -(2n-1) × b ) + ( 1-(18/37)n ) × b = ( 1-(38/37)n ) × b.
