Question:
Construct a 10 digit number in which the first digit is equal to the number of zeroes in the number, the second digit is equal to the number of ones in the number, and so on. An example of a 4 digit number of this type is 1210 - this has 1 zero, 2 ones, 1 two and 0 threes.

Answer:
6210001000
